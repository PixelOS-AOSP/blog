# March 2026 Update

This month, we've incorporated the March security patch from `android-security-16.0.0_r4`, into Android 16 QPR2. Along with that, we’ve been focusing on improving the system and user experience.

<Banner src="https://raw.githubusercontent.com/PixelOS-CI/blog_assets/refs/heads/main/2026-03-09/banner.webp" />

As always, detailed changelogs can be found on our [changelogs page](https://blog.pixelos.net/changelogs).

## Other changes
- The SetupWizard has been updated with proper Material Design theming, bringing an enhanced and more consistent user experience.
- We've also fixed an issue where the weather clock style wasn't applying.
